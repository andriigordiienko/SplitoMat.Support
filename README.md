# SplitoMat Support

This is where you report problems and suggest improvements for **SplitoMat** — the fair, private expense-splitting app.

## How to get help

- 🐞 **Found a bug?** → [Report a bug](https://github.com/andriigordiienko/SplitoMat.Support/issues/new?template=bug_report.yml)
- 💡 **Have an idea?** → [Request a feature](https://github.com/andriigordiienko/SplitoMat.Support/issues/new?template=feature_request.yml)
- 💬 **A question or something to discuss?** → [Discussions](https://github.com/andriigordiienko/SplitoMat.Support/discussions)
- 🔒 **Something private** (account deletion, personal data / GDPR, payments)? → email **splitomate@gmail.com**

The fastest way to file a bug is from inside the app: **Settings → Help & feedback → Report a bug** — it pre-fills your app version and device so we can reproduce it quickly.

## Please note

Issues and discussions here are **public**. Don't post sensitive information — full names, email addresses, amounts, or receipt photos. For anything private, use the email above.

## Before you post

- Search [existing issues](https://github.com/andriigordiienko/SplitoMat.Support/issues?q=) and [discussions](https://github.com/andriigordiienko/SplitoMat.Support/discussions) — someone may have raised it already. A 👍 on an existing request helps us prioritise.
